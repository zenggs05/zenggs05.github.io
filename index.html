Module 1: Linear Regression
Code Implementation

import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

Generate synthetic data

np.random.seed(0)
X = 2 * np.random.rand(100, 1)
y = 4 + 3 * X + np.random.randn(100, 1)

Create and train linear regression model

lin_reg = LinearRegression()
lin_reg.fit(X, y)

Predictions

X_new = np.array([[0], [2]])
y_predict = lin_reg.predict(X_new)

Calculate mean squared error

y_pred = lin_reg.predict(X)
mse = mean_squared_error(y, y_pred)
print(“Mean Squared Error:”, mse)

Visualization

plt.scatter(X, y)
plt.plot(X_new, y_predict, ‘r-‘, linewidth=2)
plt.title(‘Linear Regression’)
plt.xlabel(‘X’)
plt.ylabel(‘y’)
plt.show()
Explanation
Data Generation:
Generated 100 random sample points using numpy
Linear relationship: y = 4 + 3X + noise
Noise follows standard normal distribution
Model Training:
LinearRegression() creates linear regression model
fit() method trains the model
Model learns parameters: intercept and slope
Prediction & Evaluation:
predict() method makes predictions on new data
Mean Squared Error (MSE) evaluates model performance
Lower MSE indicates better model fit
Visualization:
Scatter plot shows original data
Red line displays fitted results
Line effectively captures data trend
Module 2: Polynomial Regression
Code Implementation

import numpy as np
import matplotlib.pyplot as plt
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import make_pipeline

Generate non-linear data

np.random.seed(0)
X = np.random.rand(100, 1) * 10
y = np.sin(X) + np.random.randn(100, 1) * 0.1

Create polynomial regression model (10th degree)

model = make_pipeline(
PolynomialFeatures(degree=10),
LinearRegression()
)
model.fit(X, y)

Generate smooth curve for plotting

X_fit = np.linspace(0, 10, 100)[:, np.newaxis]
y_fit = model.predict(X_fit)

Visualization

plt.scatter(X, y, s=10)
plt.plot(X_fit, y_fit, ‘r-‘, linewidth=2)
plt.title(‘Polynomial Regression (degree=10)’)
plt.xlabel(‘X’)
plt.ylabel(‘y’)
plt.show()
Explanation
Data Generation:
Generated non-linear data: y = sin(X) + noise
Gaussian noise with standard deviation 0.1
Data range: X in [0, 10] interval
Feature Transformation:
PolynomialFeatures(degree=10) converts features to 10th-degree polynomial
Example: X → [1, X, X², …, X¹⁰]
Increases model complexity to fit non-linear relationships
Model Training:
Uses pipeline to combine feature transformation and linear regression
Model learns polynomial coefficients
High-degree polynomial can fit complex curves
Overfitting Risk:
10th-degree polynomial is overly complex and prone to overfitting
Especially problematic with small datasets or high noise
Visualization shows excessive curve fluctuations
Module 3: Ridge Regression
Code Implementation

import numpy as np
import matplotlib.pyplot as plt
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import Ridge
from sklearn.pipeline import make_pipeline

Use same data as polynomial regression

np.random.seed(0)
X = np.random.rand(100, 1) * 10
y = np.sin(X) + np.random.randn(100, 1) * 0.1

Create Ridge regression model (10th degree + L2 regularization)

model = make_pipeline(
PolynomialFeatures(degree=10),
Ridge(alpha=1.0)
)
model.fit(X, y)

Generate smooth curve for plotting

X_fit = np.linspace(0, 10, 100)[:, np.newaxis]
y_fit = model.predict(X_fit)

Calculate mean squared error

y_pred = model.predict(X)
mse = mean_squared_error(y, y_pred)
print(“Mean Squared Error:”, mse)

Visualization

plt.scatter(X, y, s=10)
plt.plot(X_fit, y_fit, ‘r-‘, linewidth=2)
plt.title(‘Ridge Regression (degree=10, alpha=1.0)’)
plt.xlabel(‘X’)
plt.ylabel(‘y’)
plt.show()
Explanation
Basic Setup:
Uses same data as polynomial regression
Same 10th-degree polynomial features
Difference: Ridge regression instead of standard linear regression
Regularization:
Ridge(alpha=1.0) creates Ridge regression model
alpha=1.0 is regularization strength parameter
L2 regularization penalizes large coefficients to prevent overfitting
Regularization Effect:
Despite using 10th-degree features, Ridge regression penalizes large coefficients
Makes model smoother and prevents overfitting
Visualization shows smoother results than standard polynomial regression
Evaluation:
Mean squared error evaluates model performance
Ridge regression typically has better generalization than standard polynomial regression
